# Seismic React

Seismic React is a package that provides React hooks to interact with the Seismic blockchain.

It is powered by [seismic-viem](/viem/intro) and [wagmi](https://wagmi.sh)

See the guides below for examples on how to get started with a wallet provider

- [RainbowKit](/react/guides/rainbowkit)
- [Privy](/react/guides/privy)
