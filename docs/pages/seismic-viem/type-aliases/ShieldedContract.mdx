Defined in: [packages/seismic-viem/src/contract/contract.ts:101](https://github.com/SeismicSystems/seismic-client/blob/63496655224ae1d3e8c312c6d8d80f51a6b6de42/packages/seismic-viem/src/contract/contract.ts#L101)

The same as viem's [GetContractReturnType](https://viem.sh/docs/contract/getContract.html#with-wallet-client), with a few differences:

- `read` and `write` use signed reads & seismic transactions
- `tread` and `twrite` behave like viem's standard read & write
