# Function: rng()

function rng(client, args): Promise<bigint>

Defined in: [packages/seismic-viem/src/precompiles/rng.ts:114](https://github.com/SeismicSystems/seismic-client/blob/658575a2378fa3042cad6a13bb56490931591f61/packages/seismic-viem/src/precompiles/rng.ts#L114)

Generates a random number using the RNG precompile.

## Example

```typescript
// Generate a random uint8
const randomUint8 = await rng(client, 1);

// Generate a random uint256
const randomUint256 = await rng(client, 32);

## Parameters

### client

[`CallClient`](/seismic-viem/type-aliases/CallClient.mdx)

The public client to use for the precompile call.

### args

`RngParams`

## Returns

`Promise`\<`bigint`\>

A promise that resolves to the randomly generated number as a bigint.

## Throws

Throws if:
  - The bytes argument is not a number or bigint.
  - The bytes argument is greater than 32.
  - The precompile call fails.
```
