# Function: rng()

```ts
function rng(client, args): Promise<bigint>
```

Defined in: [packages/seismic-viem/src/precompiles/rng.ts:134](https://github.com/SeismicSystems/seismic-client/blob/9fcfa5c0984a89e84539dc512bd70bb63c77e3ef/packages/seismic-viem/src/precompiles/rng.ts#L134)

Generates a random number using the RNG precompile.

## Example

```typescript
// Generate a random uint8
const randomUint8 = await rng(client, { numBytes: 1 })

// Generate a random uint256
const randomUint256 = await rng(client, { numBytes: 32 })

// Generate a random number with personalization
const randomWithPers = await rng(client, {
  numBytes: 16,
  pers: '0x1234567890abcdef',
})
```

## Parameters

### client

[`CallClient`](/seismic-viem/type-aliases/CallClient.mdx)

The public client to use for the precompile call.

### args

`RngParams`

The parameters for random number generation.

- `numBytes` {bigint | number} - The number of bytes to generate (must be less than or equal to 32).
- `pers` {Hex | ByteArray} [optional] - Personalization string to seed the random number generator.

## Returns

`Promise`\<`bigint`\>

A promise that resolves to the randomly generated number as a bigint.

## Throws

Throws if:

- The numBytes argument is not a number or bigint.
- The numBytes argument is greater than 32.
- The precompile call fails.
