# Function: aesGcmDecrypt()

```ts
function aesGcmDecrypt(client, args): Promise<string>
```

Defined in: [packages/seismic-viem/src/precompiles/aes.ts:228](https://github.com/SeismicSystems/seismic-client/blob/658575a2378fa3042cad6a13bb56490931591f61/packages/seismic-viem/src/precompiles/aes.ts#L228)

Decrypts ciphertext using AES-GCM decryption through a precompile contract.

## Example

```typescript
const plaintext = await aesGcmDecrypt(client, {
  aesKey: '0x000102030405060708090a0b0c0d0e0f101112131415161718191a1b1c1d1e1f',
  nonce: 123456,
  ciphertext: '0x7b226d657373616765223a22656e63727970746564227d',
})
```

## Parameters

### client

[`CallClient`](/seismic-viem/type-aliases/CallClient.mdx)

The public client to use for the precompile call.

### args

[`AesGcmDecryptionParams`](/seismic-viem/type-aliases/AesGcmDecryptionParams.mdx)

The parameters for the AES-GCM decryption operation.

- `aesKey` {Hex} - The AES key in hexadecimal format to use for decryption.
- `nonce` {number} - The nonce value to use for decryption.
- `ciphertext` {Hex} - The ciphertext in hexadecimal format to decrypt.

## Returns

`Promise`\<`string`\>

A promise that resolves to the decrypted plaintext as a string.

## Throws

May throw if parameter validation fails or if the precompile call fails.
