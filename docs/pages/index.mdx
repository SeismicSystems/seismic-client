# Seismic Typescript Client

This contains documentation for the Seismic Typescript Client. The source code can be found [here](https://github.com/SeismicSystems/seismic-client).

The Seismic client includes two packages, [Seismic Viem](./viem/intro.mdx) and [Seismic React](./react/intro.mdx).

## Seismic Viem

`seismic-viem` is a typescript package that extends [viem](https://viem.sh). It adds support for interacting with the Seismic blockchain

## Seismic React

`seismic-react` is a typescript package that composes with [wagmi](https://wagmi.sh). It provides React hooks for interacting with the Seismic blockchain.
