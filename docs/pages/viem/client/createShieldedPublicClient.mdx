# createShieldedPublicClient

## Import

```ts
import { createShieldedPublicClient } from 'seismic-viem'
```

## Usage

Creates a public client for the Seismic network with additional privacy-preserving features.

```ts
import { createShieldedPublicClient, seismicDevnet } from 'seismic-viem'
import { http } from 'viem'

const client = await createShieldedPublicClient({
  transport: http(),
  chain: seismicDevnet,
})

const teePublicKey = await client.getTeePublicKey()
```

## Return Value

A ShieldedPublicClient instance that extends viem's PublicClient with additional methods for privacy-preserving operations.

## Parameters

### parameters

The same parameters passed into viem's [createPublicClient](https://viem.sh/docs/clients/public.html).

```ts
const client = await createShieldedPublicClient({
  transport: http(), // [!code focus]
  chain: seismicDevnet, // [!code focus]
})
```

#### transport

- **Type:** `Transport`

The transport layer to use (e.g., an HTTP transport).

#### chain

- **Type:** `Chain | undefined`

The chain configuration to target (e.g., `seismicDevnet`).

## Shielded Public Actions

In addition to all standard viem PublicClient methods, the following Seismic-specific methods are available:

### getTeePublicKey

```ts
const teePublicKey = await client.getTeePublicKey()
```

Retrieves the public key of the Trusted Execution Environment (TEE).

### rng

```ts
const randomValue = await client.rng({ max: 100n })
```

Generates a secure random number. See [RNG Precompile](/viem/precompiles/rng) for more details.

### ecdh

```ts
const result = await client.ecdh({ publicKey: '0x...' })
```

Performs Elliptic Curve Diffie-Hellman key exchange. See [ECDH Precompile](/viem/precompiles/ecdh) for more details.

### AES-GCM

#### Encryption

```ts
const encryptedData = await client.aesGcmEncryption({
  data: 'text to encrypt',
  key: '0x...',
})
```

Encrypts data using AES-GCM encryption.

#### Decryption

```ts
const decryptedData = await client.aesGcmDecryption({
  data: '0x...',
  key: '0x...',
})
```

Decrypts AES-GCM encrypted data.

See [AES Precompile](/viem/precompiles/aes) for more details.

### hdfk

```ts
const derivedKey = await client.hdfk('0x...')
```

Derives a key from the initial key material. See [HDKF Precompile](/viem/precompiles/hdfk) for more details.

### SECP256K1

```ts
const signature = await client.secp256k1({
  sk: '0x123...', // Your secret key
  message: 'Hello, world!', // The message to sign
})
```

Signs a message using SECP256K1. See [SECP256K1 Precompile](/viem/precompiles/secp256k1) for more details.
