# Devnet

An alias for [`seismicDevnet1`](/viem/chains/devnet1)

## Import

```ts
import { seismicDevnet } from 'seismic-viem'
```

## Usage

`seismicDevnet` is simply an alias for [`seismicDevnet1`](/viem/chains/devnet1). It provides a default chain configuration for connecting to the Seismic development network.

```ts
import { seismicDevnet } from 'seismic-viem'
import { createPublicClient, http } from 'viem'

const client = createPublicClient({
  chain: seismicDevnet,
  transport: http(),
})
```

## Description

This variable exports the same chain configuration as [`seismicDevnet1`](/viem/chains/devnet1), providing a convenient default when you don't need to specify a particular devnet node.

The Seismic development network runs at `node-1.seismicdev.net` with all the same properties as described in the [`seismicDevnet1`](/viem/chains/devnet1) documentation.
