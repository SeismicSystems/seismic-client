# Devnet 2

Pre-configured chain for connecting to the Seismic development network at node-2.seismicdev.net

## Import

```ts
import { seismicDevnet2 } from 'seismic-viem'
```

## Usage

`seismicDevnet2` provides a pre-configured chain for connecting to the first Seismic development network node.

```ts
import { seismicDevnet2 } from 'seismic-viem'
import { createPublicClient, http } from 'viem'

const client = createPublicClient({
  chain: seismicDevnet2,
  transport: http(),
})
```

## Description

This is a pre-configured chain for connecting to the Seismic development network running at `node-2.seismicdev.net`.

- **Network Type**: Single-node development network
- **Block Explorer**: Available at `explorer-2.seismicdev.net`
- **Implementation**: Runs on Seismic's fork of reth in development mode

## Properties

The `seismicDevnet2` chain configuration contains the following properties:

### id

- **Type:** `number`
- **Value:** `5124`

The chain identifier.

### name

- **Type:** `string`
- **Value:** `"Seismic"`

Human-readable name of the chain.

### nativeCurrency

- **Type:** `object`
- **Value:** `{ decimals: 18, name: "Ether", symbol: "ETH" }`

The native currency configuration of the chain.

### rpcUrls

- **Type:** `object`
- **Value:**
  ```ts
  {
    default: {
      http: ["https://node-2.seismicdev.net/rpc"],
      webSocket: ["wss://node-2.seismicdev.net/ws"]
    }
  }
  ```

Collection of RPC endpoints for connecting to the local Anvil instance.

### formatters

- **Type:** `ChainFormatters`
- **Value:** `seismicChainFormatters`

Custom formatters for Seismic-specific data structures.
